CSS - Cascading Style Sheet


O que é CSS e por que utilizar:

CSS é uma linguagem de estilos que serve para compor a parte visual das páginas na internet. Um bom motivo para utilizarmos CSS é a praticidade que nos oferece ao podermos definir um ou estilo unico (evitando repetição de código) ou  arquivo separado contendo as configurações de estilo desejadas, deixando o código HTML menos poluído e, neste último caso permitindo reutilização da folha de estilos em outro código, além de nos permitir opções mais variadas de personalização.



Diferentes modos possíveis de utilização de CSS e suas diferenças:

  • Estilo inline: utilizado na própria tag, sendo criado individualmente em cada tag separadamente (atributo style).
  • Estilo incorporado: no inicio do cógido, definimos a lista de estilos que serão utilizados no mesmo e as tags as quais esses estilos serão aplicados (tag style).
  • Estilo vinculado: os estilos que utilizaremos no código html são criados em outro documento (.css), e este outro documento é linkado dentro do código, diminuindo assim a poluíção do código e permitindo reutilização em outros códigos.

A diferença entre eles é basicamente se o estilo será aplicado em cada tag individualmente (1), no inicio do documento definindo quais tags usarão determinado estilo (2) ou se será feito uma "folha de estilos" e outro documento linkado ao código principal.
